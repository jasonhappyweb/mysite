Line Bot
自動報名系統

整合 LINE Messaging API 與 Hangfire 排程，打造自動化報名、繳費提醒的完整流程。

📩 使用者輸入代碼 → 自動回應「報名」提示 → 系統 API 接收 Line ID 與名稱
✅ 報名成功：顯示使用者資訊與報名狀態，記錄時間與費用
⏰ 活動前自動提醒：Hangfire 定時推播提醒使用者繳費
💵 使用者完成繳費後，系統自動更新狀態為「已繳交」
🛠️

使用技術

🔗LINE Messaging API
📩Webhook 綁定
📅Hangfire 排程
🌐ASP.NET Core MVC
🗄️PostgreSQL
☁️Render 雲端部署
📦GitHub CI/CD
