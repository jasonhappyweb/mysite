Och 管理系統 Demo 展示

使用技術

  • ✅ ASP.NET Core MVC 架構設計
  • ✅ 前端整合 Bootstrap Modal、SweetAlert2、jQuery
  • ✅ 後端資料存取：Entity Framework Core + Dapper
  • ✅ 資料庫：PostgreSQL
  • ✅ RESTful API 與 Swagger 文件

部署流程

  • 📦 GitHub 儲存程式碼與自動推送版本
  • 🐳 Docker 建立容器映像（Dockerfile）
  • ☁️ Render 免費雲平台自動部署並運行
  • 🔁 Webhook 整合 Make 自動接收外部資料並串接 Regic 系統
